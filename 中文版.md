# JLC 3D Viewer Proxy

嘉立创 3D 模型查看器（https://3d-viewer.jlc.com）的接口逆向成果 + 本地 HTTP 代理服务。

**核心能力**：登录一次后，通过纯 HTTP 接口无限次数上传模型文件并获得可分享的预览链接。


## 逆向发现

### 5 次限制机制

未登录用户只能预览 5 次。经本地存储（localStorage / sessionStorage / cookie）全面排查及前端 JS 静态分析，确认：

- **计数在服务端记录**，按访问者身份（账号/IP）区分
- 前端代码中 `countPreviewNum` 仅为事后统计上报，**无任何拦截逻辑**
- **本地无法清空**

### 登录态差异

| 行为 | 未登录 | 登录 |
|------|--------|------|
| `windowLogin` | 401 | 200 返回 JWT |
| 请求 `token` header | 无 | JWT |
| `countPreviewNum` | ✅ 被调用 | ❌ 不调用 |

登录后服务端跳过整个计数机制 → 天然无限次数。

### 完整接口架构

后端基址：`https://api.forface3d.com/forface`

**核心上传链路**（已完整逆向并复现）：

```
genUploadFileUrl → OSS 直传(PUT) → updateFileStatus → uploadModel → shareModel → previewShareModel → 分享URL
```

### 两个关键签名算法

**1. 接口签名**（`getWebSystemConfig` / `genUploadFileUrl`）：

```js
sign = MD5( requestId + "NjVkczQxczRkNTZBQVMxRDU2MWQwMzJBQTFTRCAzMmExZDU2czFkMzFTMEQzc2RT" )
requestId = Date.now().toString()
```

**2. OSS 直传签名**（关键：`content-type` 必须为空字符串）：

```
PUT
Content-MD5: <文件MD5 (Base64)>
Content-Type:                     ← 必须为空字符串
Content-Disposition: attachment; filename*=UTF-8''<文件名>
token: <JWT>
```

### 登录鉴权链

```
passport.jlc.com CAS 登录（扫码/账密）
  → cookie: PROD-JLC-CAS-SID + tgc
  → windowLogin(appId=JLC_FORFACE&code=<票据>)
  → 返回 JWT token
  → 后续所有请求带 token header
```

### 分享预览链接

```
https://3d-viewer.jlc.com/share?modelId=<id>&tokenKey=<key>
```

24 小时有效，可在任何浏览器打开（含无痕）。


## 快速开始

```bash
# 1. 安装依赖（首次含 Chromium 浏览器下载）
npm install

# 2. 启动
npm start
# → http://localhost:3721
```

## 使用流程

1. 打开 `http://localhost:3721`
2. 点击 **JLC 登录** → 弹出浏览器扫码/账密登录 → **浏览器自动关闭**（登录凭证已保存）
3. 选择模型文件上传 → 等待 5-60 秒 → 获得预览链接，**自动在新窗口打开**
4. 重复上传，无限次数

## HTTP API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/status` | 查询登录状态 |
| GET | `/login?wait=1` | 弹浏览器登录（SSE 实时推送进度） |
| POST | `/upload` | 上传文件（multipart field: `file`），返回预览链接 |
| POST | `/logout` | 清除本地凭证 |

### 接入示例

```python
import requests

# 上传（假设已登录）
with open('模型.SLDPRT', 'rb') as f:
    r = requests.post('http://localhost:3721/upload', files={'file': f})
print(r.json())  # { ok, modelId, shareUrl, ... }
```

```bash
# 命令行
curl -X POST http://localhost:3721/logout              # 退出
curl "http://localhost:3721/login?wait=1"               # 登录
curl -F "file=@模型.SLDPRT" http://localhost:3721/upload # 上传
```

### 上传响应示例

```json
{
  "ok": true,
  "file": "eye.SLDPRT",
  "modelId": "2070790253148254208",
  "shareUrl": "https://3d-viewer.jlc.com/share?modelId=2070790253148254208&tokenKey=fbdc34eb...",
  "message": "上传成功，预览链接已生成"
}
```

## 架构

```
浏览器登录（仅一次）          纯 HTTP 上传（无限次）
     │                              │
     ▼                              ▼
┌──────────┐                  ┌───────────┐
│ Playwright │ 抓取 token     │ JlcClient │  纯 Node HTTP
│ 弹窗登录   │──────────────▶│ sign 签名 │  genUploadFileUrl
│ 关闭浏览器 │ 写入 creds.json │ OSS 直传  │  OSS PUT
└──────────┘                  │ 分享链接   │  uploadModel
                              └─────┬─────┘
                                    │
                              ┌─────▼─────┐
                              │  Express   │  HTTP 服务
                              │  /upload   │  端口 3721
                              └───────────┘
```
